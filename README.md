
# Expense Tracker

This is a simple web-based Expense Tracker application that allows users to manage their income and expenses effectively.






## Features


Add Transactions: Input details such as date, description, category, and amount to add transactions.

View Transactions: Display a list of all added transactions.

Delete Transactions: Remove unwanted transactions from the list.

Summary Section: Automatically calculates and displays total income, total expenses, and net savings.

## Files

index.html

Provides the structure of the Expense Tracker application.

Includes input fields for date, description, category, and amount, along with a button to add transactions.

2. style.css

Contains the styling for the application.

Implements a clean, modern design with responsive layout.

3. script.js

Handles the functionality of the Expense Tracker.

Includes logic for adding, deleting, and rendering transactions, as well as updating the summary section dynamically.
## How to Use

Open the index.html file in any web browser.

Fill out the fields for:

Date

Description (e.g., "Groceries")

Category (e.g., "Food")

Amount (use positive values for income and negative values for expenses).

Click the "Add Transaction" button to add the transaction to the list.

View the transaction in the "Transactions" section.

Check the "Summary" section to see updated totals for income, expenses, and net savings.

To delete a transaction, click the "Delete" button next to it.
